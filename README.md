## 🚀 How to Deploy to GitHub Pages

### 1. Create GitHub Repository
```bash
# In your terminal
cd /home/megat/threads-privacy-policy
git init
git add .
git commit -m "🎉 feat: initial commit - privacy policy and terms"
```

### 2. Create Repo on GitHub
1. Go to https://github.com/new
2. Repository name: `threads-privacy-policy`
3. Set to **Public** (required for free GitHub Pages)
4. Do NOT initialize with README (we already have files)
5. Click **"Create repository"**

### 3. Push to GitHub
```bash
git remote add origin https://github.com/YOUR_USERNAME/threads-privacy-policy.git
git branch -M main
git push -u origin main
```

### 4. Enable GitHub Pages
1. Go to your repo on GitHub
2. Click **Settings** → **Pages** (left sidebar)
3. Under **Source**, select **"Deploy from a branch"**
4. Branch: **main** / **/ (root)**
5. Click **Save**

### 5. Get Your URLs
After 1-2 minutes, your pages will be live at:
- **Home**: `https://YOUR_USERNAME.github.io/threads-privacy-policy/`
- **Privacy Policy**: `https://YOUR_USERNAME.github.io/threads-privacy-policy/privacy-policy.html`
- **Terms**: `https://YOUR_USERNAME.github.io/threads-privacy-policy/terms.html`

### 6. Add to Meta Developer App
1. Go to https://developers.facebook.com/apps/
2. Select your app
3. **App Settings** → **Basic**
4. Fill in:
   - **Privacy Policy URL**: `https://YOUR_USERNAME.github.io/threads-privacy-policy/privacy-policy.html`
   - **Terms of Service URL**: `https://YOUR_USERNAME.github.io/threads-privacy-policy/terms.html`
5. Click **Save Changes**
